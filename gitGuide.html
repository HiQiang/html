git 安装：

windows:

CentOS:yum install git（一键安装）

配置：

git config --global user.name 'Your Username'（github网站的用户名）

git config --global user.email 'Your Email'（github网站的邮箱）

查看配置是否生效：

git config --list

常用命令

git clone 仓库名

git add filename（将filename切换为 . 表示当前文件夹所有文件）

git status

git commit filename

git push（上传到github仓库）

git pull (将别人更新后的仓库文件 拉 到本地)
